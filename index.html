Buscador EAN
Sube tu Excel (.xlsx) o CSV con columnas como mínimo: EAN, ID (identificación rápida) y Ref11. Si no tienes Ref11 pero sí Departamento (3), Familia (3) y Barra (5), la app la calcula automáticamente.
Para mostrar foto, añade una columna con URL de imagen (p. ej. “Foto”, “Imagen”, “Image URL”).
Si varios productos comparten los 4 dígitos finales, aparecerán todas las coincidencias.
Funciona en móviles. Puedes “Instalar” para acceso rápido. Los datos cargados se guardan en este navegador.
